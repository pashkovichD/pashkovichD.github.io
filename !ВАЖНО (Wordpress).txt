1. Не работает шаблон для записи произвольного типа?
Не забывайте после регистрации произвольного типа записи отправляться в 
админку => настройки => постоянные ссылки и нажать кнопочку "сохранить изменения"

2. Как добавить id и класс для формы Contact Form 7?
Для добавления любого идентификатора и класса в форме используются атрибуты html_id и html_class в теге, 
который вставляем на страницу WordPress или в сайдбар  

[contact-form-7 id="2028" title="Contacts" html_id="my-id" html_class="my-class"]

3. Отправку почты организовал с помощью wp_mail() (из плагина). Чтобы всё работало, нужео !Обязательно устаносить плагин WP Mail SMTP и выбрать метод отправки почты (PHP), а также обязательно в настройках сайта указать почтовый ящик с нашим доменом (например, info@домен.by)
Пример PHP-кода...

$msg = "<div style='width: 230px; padding: 20px 0 0 20px;'><p style='margin: 0 0 7px 0; padding: 5px; border-bottom: 1px dashed #aaaaaa; font-weight: bold; background-color: #f2f2f2;'>" . $nl_array['form__name'] . "</p><p style='margin: 0; padding: 0;'>" . $nl_array['form__phone'] . "</p></div>";

$headers = array(
	'From: NL-Company',
	'Content-type: text/html; charset=utf-8'
);

wp_mail('info@nlcompany.by', $nl_array['form__name'] . " (заявка)", $msg, $headers);

4. Не работали два цикла WP_Query на одной странице (использовались для вывода двух разных типов записей). 
Нужно было после каждого цикла прописать... <?php wp_reset_postdata(); ?>
см. документацию (Возвращает глобальную переменную $post в правильное состояние: в соответствие с текущим. Функцию нужно использовать каждый раз после запуска произвольного цикла. Т.е. в случаях, когда на странице есть дополнительный цикл WordPress с использованием глобальной переменной $post).
