1. Не работает шаблон для записи произвольного типа?
Не забывайте после регистрации произвольного типа записи отправляться в 
админку => настройки => постоянные ссылки и нажать кнопочку "сохранить изменения"

2. Как добавить id и класс для формы Contact Form 7?
Для добавления любого идентификатора и класса в форме используются атрибуты html_id и html_class в теге, 
который вставляем на страницу WordPress или в сайдбар  

[contact-form-7 id="2028" title="Contacts" html_id="my-id" html_class="my-class"]


3. Slick.js. Чтобы отменить дерганную перемотку, нужно отменить анимацию для элемента .slick-track
.slick-track {
	transition: none; // cancel
}

4. Slick.js. Проблема: На сайте есть два слайдера на разных страницах. При обновлении файла .js или не работает свойство rows в слайдере, где нужно на маленьких экранах 1 строка в слайдере, а на больших 2 строки, или не срабатывает JS-код (нажатие на кнопку click()).
Долго не мог понять почему.
Анализ кода показал, что при обновленни (даже просто при пересохранении) файла .js, в слайдере, где не нужно менять количество строк, в зависимости от ширины экрана, slick.js добавляет дополнительную обертку div, которая и не дает нажать (click()) на кнопку.
Чтобы решить эту проблему, нужно в настройках слайдера указать rows: 0

например, так...

$('.tabs__tabs-wrapper').not('.slick-initialized').slick({			
	infinite: true,
	arrows: false,
	dots: false,
	slidesToShow: 2,
	slidesToScroll: 2,
	draggable: true,
	rows: 0,
	variableWidth: true,
	responsive: [{
		breakpoint: 4000,
		settings: "unslick"
	}, {
		breakpoint: 991,
		settings: {
			slidesToShow: 2,
			slidesToScroll: 2
		}
	}]
});

Теперь всё работает!


5. Отправку почты организовал с помощью wp_mail() (из плагина). Чтобы всё работало, нужео !Обязательно устаносить плагин WP Mail SMTP и выбрать метод отправки почты (PHP), а также обязательно в настройках сайта указать почтовый ящик с нашим доменом (например, info@домен.by)
Пример PHP-кода...

$msg = "<div style='width: 230px; padding: 20px 0 0 20px;'><p style='margin: 0 0 7px 0; padding: 5px; border-bottom: 1px dashed #aaaaaa; font-weight: bold; background-color: #f2f2f2;'>" . $nl_array['form__name'] . "</p><p style='margin: 0; padding: 0;'>" . $nl_array['form__phone'] . "</p></div>";

$headers = array(
	'From: NL-Company',
	'Content-type: text/html; charset=utf-8'
);

wp_mail('info@nlcompany.by', $nl_array['form__name'] . " (заявка)", $msg, $headers);
